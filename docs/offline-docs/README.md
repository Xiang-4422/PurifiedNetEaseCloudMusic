# 网易云音乐 API Enhanced 离线文档

这是 `https://neteasecloudmusicapienhanced.js.org/#/` 的本地离线副本。

## 直接打开

打开 `index.html` 即可阅读，不需要联网，也不需要启动本地服务。这个版本把 `home.md` 内容嵌入到 `assets/docs-content.js`，因此不会通过浏览器请求远程 Markdown。

## Docsify 版

如果想使用更接近原站的 Docsify 运行方式：

```sh
cd docsify
python3 -m http.server 8080
```

然后访问 `http://127.0.0.1:8080/`。这个版本依赖本目录内的 JS/CSS 文件，已去掉广告、统计和远程首页地址。

## 更新来源

文档内容来自 GitHub 仓库 `NeteaseCloudMusicApiEnhanced/api-enhanced` 的 `public/docs` 目录，下载时间为 2026-04-30。
